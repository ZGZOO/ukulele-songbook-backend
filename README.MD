## Backend for Ukulele Songbook
### Tools used: 
- Node.js
- Express.js
- MongoDB
- mongoose
### Deployed on Heroku
- [live API link](https://ukulele-songbook-api.herokuapp.com/songs)
- Router (table below)

Action | Route
------ | ------
Get all songs | /songs
Get a song | /songs/:id
Create a song | /songs
Edit a song | /songs/:id
Delete a song | /songs/:id
Seed songs | /songs/seed
